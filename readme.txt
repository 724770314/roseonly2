1.数据库 "localhost","root","root"

2.login.html register.html连接的是flowerdb.sql table:vip

3.点击index.html直接进入首页，点击register.html进入注册页面,注册账户跳转到index.html（首页）,输入注册过的账号和密码直接进入首页；

4.首页中的导航划过有二级菜单的效果；

5.右边分别固定了微信二维码、在线客服和回到顶部三个模块，回到顶部按钮必须等页面滚动一段距离才出现；

6.点击导航栏任意一项，进入lovegift.html(商品列表页面)，点击任意商品进入details.html（商品详情页面）；

7.点击商品详情页面的加入购物车按钮，可进入shopcar.html（购物车页面），购物车页面对商品数量进行加减，总价也发生变化；
